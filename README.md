# Amazon-analysis-of-data
SQL project analyzing an e-commerce grocery platform's customers, products, orders, and reviews — covering DDL/DML, joins, subqueries, normalization, and business insights (MySQL).

#Problem Statement

Amazon Fresh needs a relational database that efficiently organizes customer profiles, product inventories, supplier information, orders, and reviews — enabling analysis of top-performing products, customer purchasing patterns, and inventory-demand alignment.
